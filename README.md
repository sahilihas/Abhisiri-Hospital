# Abhisiri Hospital Website

A modern, responsive hospital website designed for **Abhisiri Hospital** to offer a seamless user experience for patients and staff. Built using HTML, CSS, JavaScript, and integrated with backend-ready architecture for future scalability.

---

## 🚀 Features

- 📱 Fully responsive UI (mobile, tablet, desktop)
- 🏥 Informative sections: About, Services, Departments, Doctors, Contact
- 📞 Integrated contact form (can be connected to email or backend)
- 💡 Clean and modern design with accessible layout
- 🌐 Ready for deployment on GitHub Pages, Netlify, or Firebase
